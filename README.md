# Supplementary Calculations

This repository contains supplementary symbolic calculations for the work:

**On the Geometric Emergence of the Electromagnetic Field in Teleparallel Spacetime**

Each file contains a specific calculation or consistency test related to the tetrad, torsion, electromagnetic field tensor, gauge behavior, Lorentz covariance, or energy-momentum tensor.

---

## Files

### `elemantary_mathematical_objects.py`

This file defines the basic mathematical objects used in the symbolic calculations.

It constructs the tetrad matrix, the metric tensor, the inverse metric, the determinant, and the torsion components.

The main purpose of this file is to check the basic geometric structure generated by the chosen tetrad ansatz.

In this file, the tetrad is defined, the metric is calculated from the tetrad, and the torsion components are obtained by differentiating the tetrad components.

This file is used as the basic starting point for the other symbolic calculations.

---

### `diffeomorphism_gauge_test.py`

This file tests the relation between temporal torsion and the electromagnetic field tensor.

First, the tetrad ansatz is defined. Then the electromagnetic field tensor (F_{\mu\nu}) is calculated from the electromagnetic potential. After that, the temporal torsion (T^0{}_{\mu\nu}) is calculated from the tetrad.

The main result checked in this file is:

```math
T^0{}_{\mu\nu} = \kappa F_{\mu\nu}.
```

The file also tests the pure gauge case:

```math
A_\mu = \partial_\mu \lambda .
```

For this case, the code checks that:

```math
F_{\mu\nu}=0
```

and

```math
T^0{}_{\mu\nu}=0.
```

This shows that a pure gauge contribution does not create physical temporal torsion.

The file also calculates coordinate-indexed torsion, the torsion vector, and the electromagnetic invariant.

---

### `lorentz_covarience_test.ipynb`

This notebook tests the local Lorentz covariance of the torsion structure.

It applies a local Lorentz transformation to the tetrad and includes the corresponding spin connection contribution.

The purpose of this notebook is to check that the transformed torsion structure behaves consistently under local Lorentz transformations.

This test is important because the electromagnetic interpretation should not depend only on one fixed tetrad frame.

---

### `calculation_of_energy_momentum_tensor_EM.ipynb`

This notebook calculates the electromagnetic energy-momentum tensor symbolically.

It defines the tetrad, the metric, the inverse metric, and the electromagnetic field tensor. Then it raises indices and calculates electromagnetic invariant quantities and energy-momentum tensor components.

The purpose of this notebook is to support the action variation and backreaction calculations.

It is mainly used to check the electromagnetic energy-momentum contribution in the Maxwell-adapted tetrad background.

---

### `Appendices/`

This folder contains the appendix PDF of the work.

The appendix document gives the detailed derivations behind the calculations in the repository.

It includes derivations for the tetrad, inverse tetrad, metric, inverse metric, torsion components, Maxwell equations, action variation, energy-momentum tensor, Reissner-Nordström normalization test, Lorentz force, Aharonov-Bohm phase, gauge tests, and local Lorentz covariance.

The appendix is the detailed theoretical reference for the symbolic files in this repository.

## Requirements

The symbolic calculations use Python, SymPy, and Jupyter.

```bash
pip install sympy jupyter
```

---

## Running the Files

Run the Python scripts with:

```bash
python elemantary_mathematical_objects.py
python diffeomorphism_gauge_test.py
```

Open the notebooks with:

```bash
jupyter notebook lorentz_covarience_test.ipynb
jupyter notebook calculation_of_energy_momentum_tensor_EM.ipynb
```
